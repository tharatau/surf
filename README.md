# surf

Experimental application to surf the internet with the goal of inversion of control.

## Getting Started

Clone the repo
```shell
git clone git@github.com:ayushmxn/surf.git
```

Install dependencies
```shell
chmod 755 ./deps.sh
./install.sh
```

Update dependencies
```shell
make update
```

```shell
./surf https://www.google.ca
```
